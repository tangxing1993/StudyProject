基于Activiti的OA案例项目

**__项目架构__**

	Maven + SpringBoot + Activiti + H2 + JPA + Thymeleaf

**__项目运行__**

	cd ${project_home}
	mvn clean spring-boot:run
	访问数据库: http://localhost/oa/h2-console   sa/123456
	访问应用: http://localhost/oa   admin/1234
